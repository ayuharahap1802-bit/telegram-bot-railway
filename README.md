# 🤖 BOLA PELANGI TELEGRAM BOT

Bot Telegram official untuk Bola Pelangi dengan fitur lengkap.

## 🚀 FITUR LENGKAP

### Core Features
- ✅ Auto Welcome Member
- ✅ Auto Post Terjadwal
- ✅ Broadcast System (Queue Based)
- ✅ Multi-level Admin
- ✅ User Tracking
- ✅ Statistics Engine

### Advanced Features
- ✅ Multi-language (ID/EN)
- ✅ Granular Permissions
- ✅ Message Templates
- ✅ Quick Replies
- ✅ Database Backup
- ✅ Activity Logs
- ✅ Feature Flags
- ✅ Health Check

## 📋 DAFTAR PERINTAH

### Public Commands
| Perintah | Deskripsi |
|----------|-----------|
| /start | Memulai bot |
| /help | Bantuan |
| /info | Info akun |
| /promo | Lihat promo |
| /stats | Statistik bot |

### Admin Commands
| Perintah | Deskripsi | Izin |
|----------|-----------|------|
| /admin | Dashboard admin | Admin+ |
| /users | Daftar user | manage_users |
| /broadcast | Kirim broadcast | can_broadcast |
| /posts | Kelola auto post | manage_posts |

### Super Admin Commands
| Perintah | Deskripsi |
|----------|-----------|
| /admins | Kelola admin |
| /settings | Pengaturan |
| /backup | Backup database |
| /logs | Lihat log |
